## 🎯 핵심 전략 요약

### 1. 시장 포지셔닝
- **싸이스트(MasterFlex 한국 총판)** = 가장 강력한 경쟁자 (브랜드 권위·유료광고)
- **바이오컨비젼** = Lead Fluid 동일 SKU 판매 중인 직접 경쟁자
- **셀렙 차별화** = "가성비 호환 라인 + 응용분야 가이드 + 호환성 매트릭스"

### 2. 키워드 전략 (브랜드 검색 부재 전제)
한국 시장에서 "리드플루이드"·"BT101S" 등 브랜드/모델 검색은 발생하지 않음. 따라서:
- **A 메인 (실험실용 정량펌프 등)** + **F MasterFlex 호환** + **C 응용분야** + **E 소모품** = 진짜 트래픽 입구
- **B 시리즈/모델 롱테일** = SEO 구조용·기존 방문자 전환용으로만

### 3. 4사 공통 빈틈 (셀렙 차별화 영역)
1. **응용분야 페이지 6개** — 5사 모두 없음
2. **호환성 매트릭스** — 5사 모두 없음
3. **유량 슬라이더 필터** — 5사 모두 없음
4. **시리즈 비교 페이지** — 5사 모두 없음
5. **MasterFlex 1:1 가성비 비교** — 싸이스트가 만들 수 없는 콘텐츠

### 4. 첫 화면 UX 핵심
- **4단계 위저드** (응용 → 제품 → 유량 → 특징)
- **결과 0개 사고 차단** (스마트 비활성화)
- **선호 환경 = 소프트 가중치** (필터 안 함)
- **카드 5개 + 응용 6개 + 유량 슬라이더 + 검색** 순서 (1280px 첫 스크린)

## 📜 지침 변경 이력 (Decision Log)

### 2026-04-29: 소모품 격상 (지침 정면 변경)

**원래 지침 (영쨩님 최초 작성)**
> "소모품(그룹 6, 7)은 카드가 아니라 하단 '소모품·교체부품' 섹션으로 분리"

**변경 결정**
- 소모품 = **8카드 메가 섹션으로 격상** (튜브·시린지·펌프헤드·피팅·바이알·병·클램프·연료전지 스택)
- 배치 = **Hero 직후 (펌프 5카드보다 위)**
- 펌프 5카드 = 그대로 유지하되 메가 섹션 다음으로 이동

**전략적 사유 (영쨩님)**
1. **트래픽 패턴 비대칭** — 소모품 재구매(70~80%) >> 펌프 신규 도입(20~30%)
2. **궁극 목표는 펌프 1페이지 진입이지만**, 그 도달 경로로는 두 가지가 있음:
   - (a) "정량펌프" 직접 검색 → 즉시 구매 (직접 경로)
   - (b) **소모품 반복 구매 → 사이트 신뢰 누적 → 추후 펌프 구매 (간접 경로)**
3. 영쨩님 분석: **(b) 경로의 누적 전환율이 (a) 경로의 단발 전환율보다 높음**.
   소모품으로 자주 방문하는 사이트가 "여기 펌프관련 다 있네"라는 신뢰를 만들고,
   결국 펌프 신규 도입 시점에도 같은 사이트를 선택하게 만드는 효과.

**의사결정자 / 일자** — 영쨩님 / 2026-04-29 채팅

**부수 변경**
- Title·Meta·H1에 "정량펌프 + 호환 소모품" 명시 (포지셔닝 양립)
- 카테고리 랜딩 = 펌프 페이지가 아니라 "**펌프 + 소모품 통합 허브**"로 재정의
- 펌프 SEO 신호는 5카드 위치 + 풀 시리즈/모델 페이지 깊이로 보존
- 의사결정자가 "A 유지"로 명시 결정 (재고 권장 ① 결과)
