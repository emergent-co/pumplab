## 공통 성공 패턴 (7가지)

1. 영문 병기 키워드 (5사 공통)
2. 카테고리 → 모델 2층 단순 구조
3. 모델 코드 URL/제목 노출
4. 전화·이메일 상단 고정
5. 카테고리 세분화로 롱테일 흡수 (바이오컨비젼·싸이스트 강함)
6. 자매/별도 사이트로 키워드 영역 분산
7. 외부 노출 자산화 (싸이스트 YouTube·박람회·유료광고, 바이오컨비젼 BRIC Bio마켓)

## 셀렙이 차별화할 빈틈 (5가지)

1. **응용분야(Application) 카드 + 응용별 랜딩** — 5사 공통 빈틈 ★ 최우선
2. **호환성 매트릭스 (튜브×헤드×시린지)** — 5사 공통 빈틈 ★
3. **유량 기반 슬라이더 필터** — 5사 공통 빈틈
4. **시리즈 비교/선택 페이지** — 5사 공통 빈틈
5. **가격·성능 정량 비교 콘텐츠** — "MasterFlex BT100 vs Lead Fluid BT101S" 호환·가성비 비교 (싸이스트가 만들 수 없는 콘텐츠)
