## 1. 루프6 — 네이버 생태계

### 1-1. 관측 실패 고지 (먼저 밝힘)

네이버 직접 관측은 **모든 경로에서 차단**되었다.

| 시도 | 결과 |
|---|---|
| Chrome → search.naver.com / kin.naver.com / smartstore | 전부 차단 |
| web_fetch → search.naver.com | HTTP 403, blocklist |
| WebSearch에 blog.naver.com 지정 | API 400 — **크롤러가 네이버 블로그를 아예 색인하지 않음** |
| 블랙키위 / 키워드마스터 | 로그인 요구 / Cloudflare 봇 차단 |

따라서 **8개 검색어의 네이버 SERP 영역 순서, 블로그 탭 실제 글, 지식iN 질문·답변자, 네이버쇼핑 가격대는 전부 "확인 불가"**다. 추정으로 채우지 않았다.

**이 표는 사람이 20~30분 직접 검색하면 채워진다.** 8개 키워드(정량펌프 / 연동펌프 / 시린지펌프 / 실험실 펌프 추천 / 페리스탈틱 펌프 / 튜빙펌프 / 방폭 정량펌프 / 리드플루이드)의 통합검색·블로그·지식iN·쇼핑 탭 스크린샷만 있으면 구조 분석과 선점 전략을 마무리할 수 있다.

### 1-2. 대신 확보한 것 — 업체별 네이버 채널 실측

업체 자사 사이트의 아웃바운드 링크를 전수 추출해 확인했다. 이건 추정이 아니라 실측이다.

| 업체 | 네이버 블로그 | 스마트스토어 |
|---|---|---|
| **바이오컨비젼** | ✅ blog.naver.com/mwoo1004 | ✅ smartstore.naver.com/konvision |
| **SCIST(사이스트)** | ✅ blog.naver.com/scist-no1 | ✅ smartstore.naver.com/scist |
| **펌프맨(TJ)** | ✅ blog.naver.com/tj_pumpmen | ❌ |
| **TDL사이언스** | ✅ blog.naver.com/tdl-sci | ✅ smartstore.naver.com/americafitting |
| 랩캠퍼스, pumpman.co.kr, 세진과학, 펌프샵, 올포랩, 코랩샵, 석림랩텍, 오창교역, 동진과학 | ❌ (9곳) | ❌ |

- 조사 13곳 중 **블로그 운영 4곳(31%), 스마트스토어 3곳(23%). 69%는 네이버 채널이 전무하다.**
- 가장 공격적인 곳은 **바이오컨비젼**. 자사 쇼핑몰 메인에 최신 블로그 글 2건을 상시 노출시키는 구조를 쓰며, 포스트 ID 자릿수로 볼 때 현재 활발히 운영 중이다.
- 1차에서 언급된 `blog.naver.com/allpumps`는 **어떤 경로로도 존재 확인 불가**.

### 1-3. 간접 증거 — 네이버에도 실사용 콘텐츠는 없다

| 질의 | 결과 | 성격 |
|---|---|---|
| 정량펌프 네이버 블로그 | 1건 | 업체 기술자료 (블로그 아님) |
| 연동펌프 블로그 후기 | 7건 | 전부 제조사·쇼핑몰·학회 PDF. **후기 0건** |
| 튜빙펌프 실험실 사용 후기 블로그 | 3건 | 전부 업체 제품페이지. **후기 0건** |
| 시린지펌프 실험실 지식인 추천 | 9건 | 전부 유통사 카탈로그 |
| "blog.naver.com" 정량펌프 | 1건 | 네이버 블로그 아님 |

총 28건 중 **비업체·실사용 콘텐츠는 1건(3.6%)**, 그것도 네이버가 아니라 **BRIC**이었다.

> **전략적 시사점**: 한국 연구자의 펌프 관련 실사용 질문은 네이버 지식iN이 아니라 **BRIC으로 흘러가고 있을 가능성**이 크다. 지식iN 공략보다 BRIC의 질문을 블로그 콘텐츠로 재구성하는 편이 효율적일 수 있다.

### 1-4. 구글 vs 네이버 — 어디가 뚫기 쉬운가

| 항목 | 구글 | 네이버 |
|---|---|---|
| 경쟁자 | pumpmen이 모델별 랜딩 10개+ 도배, Watson-Marlow·ProMinent 등 고권위 글로벌 도메인 | 블로그 운영 **4곳뿐** |
| 진입 난이도 | 높음 | **낮을 것으로 추정** |

**네이버가 뚫기 쉽다.** 다만 이 결론은 SERP를 직접 못 본 상태의 추정이며, 바이오컨비젼의 블로그 운영 밀도에 따라 헤드 키워드 난이도는 달라질 수 있다.

### 1-5. 네이버 선점 가설 TOP 5 (검증 필요)

1. BRIC의 실사용 질문을 네이버 블로그 포스트로 재구성 — 무경쟁 가능성 높음
2. **"방폭 정량펌프"** — 자사 페이지 보유는 2곳뿐, **블로그 콘텐츠는 0건**
3. **리드플루이드 모델명 단위 블로그** — pumpmen이 구글은 장악했으나 **네이버 블로그 콘텐츠는 없음**
4. **비교·선택 가이드** — 6개 질의 전부에서 0건
5. 네이버 채널이 없는 69% 업체군의 취급 제품 키워드

**개별 판정**

- **Thomas Pumps**: H2 8개 중 7개가 완전 질문형("어떻게 작동합니까?", "얼마나 오래 지속됩니까?"). 각 H2가 검색 질의와 1:1 매칭. **질문형 구조의 최고 사례.**
- **Ossila**: 유일하게 **DOI 논문 2편을 각주 인용**. "측정 시스템은 가장 부정확한 요소만큼만 정확하다" 같은 격언형·규칙형 문장 다수. **인용 가치 최고.**
- **Cole-Parmer**: 번호형 소제목 + 표 1개 + 단위 붙은 숫자. 청크로 잘라 쓰기 좋은 교과서형.
- **LeadFluid**: 형식적으로 **최약체**. H2 0개, 표 0개, **수치 0개**, 600단어. 질문형 타이틀과 최신 날짜로 노출만 겨우 잡는 유형. 저자가 `root`(익명).
- **Watson-Marlow**: **본문이 빈 응답으로 반환됨**(JS 렌더링 + 봇 차단). 세계 1위 브랜드가 **LLM 인용 후보에서 구조적으로 배제**되고 있다. 경쟁사가 비운 자리다.

### 2-4. 콘텐츠 1편 작성 템플릿

**목표 규격**: 한글 **5,500–7,000자**, **H2 6–7개 + H2당 H3 2–3개**, 표 **2–3개**, 인용 가능 단정문 **35개 이상**, 수치 **25개 이상**

```
【0】 헤드 — 질문형 title + 연도 / meta에 숫자 1개 / 발행·수정일 노출 /
     저자 실명+직함+경력 1줄 / breadcrumb 3단
【1】 리드 문단 — 한글 150자 이내, 결론부터
     ① 정의 1문장 ② 선정 기준 3가지를 숫자와 함께 ③ 범위 1문장
【2】 "이 글에서 다루는 내용" 목차 6–7줄 (H2와 동일 문구)
【3】 H2-1 질문형: "~는 어떻게 작동하나요?" + 구성부품 불릿 6–8개 + 다이어그램
【4】 H2-2 질문형: "~와 ~는 무엇이 다른가요?" ★비교표 1 + 표 요약 문장 2–3개
【5】 H2-3 질문형: "어떤 기준으로 고르나요?" H3를 번호 매김 6개, 각 300–450자
【6】 H2-4 명사형: "재질별 특성과 선택 기준" ★표 2 + 외부 출처 1–2건
```

### 3-4. 캘리브레이션 실무

### 4-5. 금액 구간별 전략

**① 300만원 미만 — "교수 지갑"**
산단을 거치지 않고 연구책임자가 연구비카드로 직접 결제. 무기는 조달자격이 아니라 **검색 노출 + 즉시 견적 + 카드결제 + 빠른 배송**. **여기서 쌓은 레퍼런스가 상위 구간 규격서에 반영된다. 손익만 보고 버리면 안 된다.**

**② 300만~2천만원 — "견적 3사 안에 들기"**
대학 거래처 등록이 절대 조건. 복수견적 원칙이나 **품목지정 사유서**로 단독 지정 가능 → 교수가 사유서를 써줄 만한 **기술적 차별점을 문서로 제공**. 1인 견적 수의계약 가능 구간이므로 **유통사 최적 스윗스팟**. 여성·장애인·사회적기업 확인서가 있으면 **한도가 5천만원으로 2.5배**.

**③ 2천만~5천만원 — "종합쇼핑몰 등재"**
MAS 등재가 없으면 매번 입찰 경쟁. **등재 시 2단계경쟁 없이 수요기관이 바로 선택** → **MAS 투자의 회수 지점.**

**④ 5천만~1억 — "2단계경쟁 + 심의 지원"**
5인 이상 가격·품질 경쟁 → **손익분기 최저단가를 사전 계산**해두고 그 아래로는 응찰 금지. 3천만원 이상이므로 장비심의 대상 — **총구축비 산정 근거, 중복투자 방지 설명자료, ZEUS 공동활용 계획서를 고객 대신 만들어 주는 것이 최고의 차별화.**

**⑤ 1억원 이상 — "예산 편성 단계 진입"**
본심의는 예산편성 단계에서 이뤄진다. 즉 **구매 1년 전에 승부가 난다.** 전년도 상반기에 연구책임자와 스펙·견적을 함께 만들어야 한다. **낙찰 시점에 영업을 시작하면 이미 늦다.**

### 4-6. 최우선 확인 사항

**펌프 세부품명이 '25~'27 중기간경쟁제품 610개 목록에 포함되는지**를 SMPP(smpp.go.kr)에서 세부품명번호로 직접 조회해야 한다. **포함된다면 직접생산확인증명서가 필수가 되고, 순수 유통사는 그 경로가 막힌다.** 이 결과에 따라 전략이 갈린다.

기타 미확인: 나비엠알오 공급사 입점 수수료율(1899-0947 문의 필요), 올포랩 셀러 입점 조건, 실험기자재 업계 실제 마진율(산업 일반론 3~30%만 확인), MAS 계약기간(조달청 "3년 원칙" vs 블로그 "최대 5년" 상충), 2026년 조달수수료 개정 여부, 출연연 개별 구매규정.

9. 가격 원 단위 공개 검토 — 경쟁 10곳 중 9곳이 견적문의형

**Week 8–14 — 채널 확장**
10. MAS 신청 (적격성 평가 20~30영업일 + 전체 2~3개월)
11. 올포랩·나비엠알오 입점
12. 네이버 블로그 개설 — 경쟁 4곳뿐, 발행 콘텐츠를 네이버용(6,000–8,000자)으로 재편집

### 5-3. 성패를 가를 3가지

1. **통설을 뒤집는 근거를 가졌는가.** 루프8에서 확인한 교정 6건은 국내 어디에도 없다. 이건 검색 순위가 아니라 **인용 가치**를 만든다.
2. **300만원 미만 구간을 지키는가.** 여기가 레퍼런스 공장이다. 마진이 얇다고 버리면 상위 구간 진입로가 함께 닫힌다.
3. **PDF와 견적문의라는 두 개의 벽을 먼저 허무는가.** 경쟁사 전원이 이 두 벽 뒤에 있다. 먼저 나오는 쪽이 가져간다.
