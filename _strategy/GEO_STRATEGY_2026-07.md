## 2. 모델별 인용 메커니즘 (전략의 전제)

| | Flash-Lite | Pro |
|---|---|---|
| 웹검색 | O — 실시간 | **X** — "실시간 웹 검색이 제한되어 있어"라고 직접 명시 |
| 출처 | 실제 웹페이지 칩 | 없음. 강제 시 제조사 공식사이트를 사후 부착 |
| 브랜드 선정 근거 | 검색된 페이지 | 학습 데이터 |
| 공략법 | **SEO·색인** | **개체각인(entity salience)** |

Pro가 꼽은 브랜드는 Masterflex·Watson-Marlow·KNF·ProMinent·SSI로 실험실 프레임이 정확하다. 반면 Flash-Lite는 웹을 긁다가 Grundfos·Iwaki 같은 산업 약품주입 쪽으로 이탈하기도 한다.

**함의:** 인용 획득은 Flash-Lite 계열에서만 가능한 목표다. Pro 계열에서는 인용이 원리적으로 불가능하며, KPI는 인용수가 아니라 **답변 본문 브랜드 언급**이어야 한다.

## 3. 검증된 발견

### 3-1. 인용 0 — 29건 전부

카테고리를 연동·시린지·기어·방폭·구매실무로 넓혀도 동일하다. 자사에 동일 주제 페이지가 존재하는 질의에서도 0이다.

### 3-2. "정량펌프"는 수처리 용어다 (재현 확인)

| 질의 | 결과 프레임 | 인용 |
|---|---|---|
| 실험실 정량펌프 추천 | 산업 약품주입 | Grundfos, Iwaki, PSG, Diener |
| pH 조절용 정량펌프 | 수처리 | Newater(6), JENSPRIMA, almawatech / 브랜드 Grundfos·Milton Roy·ProMinent·SEKO |
| **배양기 배지 자동 공급 펌프** | **실험실** | 인용 0이나 **연동펌프를 1순위 추천** |

한국어 웹에서 "정량펌프"의 주인은 수처리 업계다. 실험실 프레임을 유지하려면 **용도어**(배양·분주·미세유체·관류)나 **형식어**(연동·시린지·기어)로 콘텐츠를 잡아야 한다.

### 3-3. 검색 트리거는 고유명사에 반응한다

| 질의 | 인용 |
|---|---|
| 연동펌프를 PC로 제어하는 방법 | **0** |
| 실험실 펌프 여러대 동시 제어와 무인 장시간 운전 | **0** |
| 실험실 펌프 **Modbus RS485** 제어 소프트웨어 | **7** (GitHub·NModbus·Modbus Tools 등) |

서술형·절차형 질의에서는 모델이 파라메트릭 지식만으로 답하고 검색을 생략한다. 인용 0 구간에서 "인용되는 것"은 도달 불가 목표다.

### 3-4. 국내 경쟁자 인용 지도

| 경쟁자 | 인용된 질의 | 채널 |
|---|---|---|
| 코랩샵 KOLAB | 가격·구매처(5회), 견적(6회) | 자사몰 |
| **레보딕스 Revodix** | 시린지펌프(4회), 미세유체, 가격, 견적 **본문 실명 언급** | **티스토리 블로그** |
| 티제이바이오앤셀 | **리드플루이드 브랜드 질의(3회)**, 가격 | 자사몰 |
| 영진·덕산·아토코리아 | Masterflex 국내 대안 | 자사몰 |
| 올포랩 | 가격·구매처 | 자사몰 (업로드 권한 보유) |

- **리드플루이드 브랜드 질의에서 최다 인용은 본사 leadfluid.com(6회)이다.** 티제이는 3위.
- 레보딕스는 도메인 권위가 아니라 **티스토리 + 꾸준한 실무 글**로 여러 질의를 가로질렀다.

### 3-5. 자사 강점이 답변 본문에 이미 등장한다 (인용만 없을 뿐)

| 질의 | AI가 한 말 | 자사 대응 |
|---|---|---|
| A/S 보증기간 | **"기본 1년, 브랜드별 1~3년"** | 3년 무상보증 = 시장 최상단 |
| A/S 보증기간 | 대리점에 물어볼 4개 질문: 보증개월·소모품포함·연장프로그램·부품재고 | 4개 전부 유리하게 답변 가능 |
| 견적 받는 법 | "A/S 보증기간·소모품 수급을 문서로 확인하라" | 국내 직접 A/S |

| **나라장터 조달** | **"공급업체에 종합쇼핑몰 제품 등록을 요청하라"** | **등록 시 즉시 매출 경로** |

## 4. 사이트 기술 감사 (Read·Grep으로 확인)

### 정상
렌더링(정적 HTML), robots.txt(GPTBot·ClaudeBot·PerplexityBot Allow), canonical, JSON-LD(Organization·Brand·FAQPage·BreadcrumbList), `_redirects` 14개 규칙.

### 확인된 실제 문제

**① 논문 사례 6편 소실 — 최우선**

`setups/*.html` 6개가 396바이트 noindex meta-refresh 스텁이며, 본문 내용이 사이트 어디에도 없다(`"Nature Electronics|Nature Communications"` 전체 검색 0건). 스텁 → `/pump/troubleshooting/`(이것도 스텁) → `/pump/guide/`로 이어지는 **meta-refresh 2홉 체인**.

원인: IA 이전 커밋이 펌프 콘텐츠를 `/pump/` 하위로 옮기면서 논문 6편만 누락하고 스텁으로 덮음. 이후 다른 커밋이 그 목적지를 또 스텁으로 덮음.

git 이력에 원본(8.2~9.3KB)이 남아 있어 **복구는 명령 한 줄**이다. 재작성 불필요.

| slug | 모델 | 저널 |
|---|---|---|
| brain-electrode-tyd01 | TYD01-01 시린지 | Nature Electronics 2024 |
| catheter-heparin-bt101 | BT101L 연동 | Nature Communications 2024 |
| co2-capture-ct3001f | CT3001F PEEK 기어 | Nature Communications 2024 |
| heart-eshp-bt101l | BT101L 연동 | Front. Cardiovasc. Med. 2021 |
| damo-recirculation-bt600s | BT600S 연동 | Environ. Sci. Technol. 2021 |
| nitrification-ph-bq50s | BQ50S 정량 | Bioresource Technology 2017 |

※ 복구 시 `co2-capture-ct3001f`의 "마그네틱펌프" 표기를 **PEEK 기어펌프**로 정정할 것.

**② sitemap 불일치** — sitemap.xml 40 URL vs 실제 콘텐츠 페이지 63개. `_build/posts.json`의 포스트가 1건만 남아 발생.

**③ 죽은 URL을 가리키는 내부 링크** — `/application/pump-selection` 14개 파일, `/application/pump-pc-control-modbus-rs485` 11개, `/application/tube-selection` 10개 등.

**④ llms.txt가 301·noindex된 구 URL을 AI에게 안내** (13개).

**⑤ 인용 그래프 고립** — 외부 아웃바운드 링크가 사실상 0(폰트 CDN·나비엠알오·유튜브 제외). 인바운드도 0. 논문 6편 복구만으로 nature.com·frontiersin.org·acs.org·sciencedirect.com 4개 권위 도메인으로 나가는 링크가 생긴다.

## 5. 전략 — 우선순위

### 1순위 · 본사 공식 대리점 등재 요청 (비용 최소, 효과 최대)

leadfluid.com은 브랜드 질의에서 6회, 튜브 관리 질의에서 5회 인용된다. **이미 이기고 있는 도메인**이다. 본사 distributor 페이지에 "Korea Authorized Distributor: 실험셋업연구소(rndsetup.com)" 한 줄을 올리는 것이, 자사 도메인을 인용되게 만드는 어떤 방법보다 싸고 빠르다. 대리점 계약상 요구 근거도 있다.

**소요:** 이메일 몇 통 · **선행조건:** 없음

### 2순위 · 나라장터 조달 등록

`나라장터 조달 구매` 질의는 인용 0인데, 답변이 사용자에게 "공급업체에 종합쇼핑몰 제품 등록을 요청하라"고 직접 지시한다. 공공기관·국립대 연구실이 주 고객군인 업종에서 **등록 여부 자체가 매출 경로**다. 콘텐츠 싸움이 아니다.

**소요:** 등록 절차 확인 후 수일 · **선행조건:** 없음

### 3순위 · 사이트 위생 복구

논문 6편 git 복구 → 2홉 체인 제거 → posts.json 재구축 → 내부 링크 최종 URL 직결 → llms.txt 갱신 → DOI 외부링크 노출.

이건 "권위 문제냐 색인 문제냐"를 판별하기 위한 **전제조건**이기도 하다. 지금 상태로는 판별 자체가 불가능하다.

**소요:** 1~2일 · **선행조건:** 없음

### 4순위 · 티스토리 채널 개설

레보딕스가 실제로 뚫은 경로다. 자사 도메인 권위를 6~12개월 기다리는 대신, 검증된 방법을 그대로 쓴다.

첫 글 3편(전부 인용 0 구간 + 자사가 사실로 유리하게 답할 수 있는 주제):
1. 실험실 펌프 A/S 보증기간 — 업계 기준과 확인할 4가지
2. 실험실 펌프 견적 받는 법 — 업체 비교 체크리스트
3. 연구비 장비 구매 규격서 작성 가이드

**소요:** 주 1편 · **선행조건:** 없음

### 5순위 · 올포랩·캐시바이 등록

업로드 권한 보유. 다만 인용 크레딧은 해당 몰 도메인으로 간다. 답변 본문에 판매처로 언급되는 값은 있으나 우선순위는 아니다.

### 보류 · 재검토 필요

| 항목 | 판단 |
|---|---|
| **GitHub에 Modbus 제어 코드 공개** | **폐기 권고.** 제어 소프트웨어는 유일한 해자이며, 인용 로그에 등장한 경쟁 유통사가 그대로 가져다 쓸 수 있다. 확률 낮은 인용과 해자를 맞바꾸는 비대칭 거래 |
