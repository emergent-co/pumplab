# 실험셋업연구소 GEO — 당장 할 일 vs 하면 안 되는 일

작성 2026-07-18 · 근거: Gemini 실측 60건(일반 29 + leadfluid 30) + GSC 색인 확인 + 사이트/호스팅 감사

## 판단 기준 (한 줄)

GSC가 **"페이지는 색인됨"**을 확인해줬다 → 문제는 콘텐츠·크롤링이 아니라 **권위**다.
그래서: **권위를 쌓거나 이미 인용되는 채널에 올라타는 일 = 한다.**
**약한 자기 도메인에 콘텐츠만 얹거나, 도달 불가능한 걸 쫓거나, 해자를 내주는 일 = 안 한다.**

## ✅ 당장 해야 할 일 (근거 직접·비용 낮음)

### 1. 논문 셋업 6편 복구 — 최우선
- **근거**: 60건 중 rndsetup.com이 인용된 **유일한 사례**(leadfluid #15, "논문에 쓴 사례"). 인용 출처가 `/setups/` 페이지(대체명 "정량펌프연구소")로 확인됨. 그런데 그 6편이 지금 **396바이트 삭제 스텁** 상태.
- **의미**: 유일하게 작동하는 인용 자산을 스스로 꺼둔 것.
- **방법**: git 이력에 원본(8~9KB) 있음. `git show 631bbfd^:setups/<slug>.html` 로 복구. 명령 한 줄.
- **주의**: 복구 시 co2-capture-ct3001f의 "마그네틱펌프" → "PEEK 기어펌프" 정정.

### 2. HTTPS 강제 — ✔ 완료 (2026-07-18)
- **근거**: GSC "HTTPS로 제공되지 않음" 경고 + 라이브에서 http가 https로 리다이렉트 안 됨.
- **처리**: 호스팅이 GitHub Pages가 아니라 **Cloudflare(존 rndsetup.com, Pages 프로젝트 pump-cellab-kr)**임을 확인. Cloudflare SSL/TLS → Edge Certificates → **"Always Use HTTPS" 토글 ON** 완료.
- **효과**: 존 전체 http→https 301 강제. Google 재크롤 시 HTTPS 플래그 해소.

### 3. www → non-www 301 — ✔ 완료 (2026-07-18)
- **근거**: GSC가 www 주소를 검사했는데 canonical은 non-www. 서버 301 없어 링크 신뢰도 분산.
- **처리**: Cloudflare Rules → Redirect Rules → "Redirect from WWW to root" 템플릿으로 룰 배포. `https://www.*` → `https://${1}` (301). **검증 완료**: www 주소가 non-www로 실제 301됨.

### 4. 본사 leadfluid.com 대리점 등재 요청
- **근거**: 본사 도메인은 품질·비교·나노리터 질의(leadfluid #6·#9·#27)에서 **단독 인용원**. 이미 이기는 도메인에 이름 올리는 게 티제이바이오앤셀 각인(#1·#2·#4·#20·#25·#30)을 깨는 최저비용 수단.
- **방법**: 본사에 "Korea distributor: 실험셋업연구소(rndsetup.com)" 등재 요청 이메일. 대리점 계약상 요구 근거 있음.

### 5. 캐시바이에 A/S·제어SW·가격 명기 등록
- **근거**: 캐시바이는 가격 질의의 **단일 소스**(leadfluid #8·#10·#14). 업로드 권한 이미 보유.
- **방법**: 리드플루이드 제품에 가격·국내 A/S·3년 보증·제어 SW를 명기해 등록 → AI가 읽는 유일한 가격 채널에 자사 개체 주입.

### 6. "국내 A/S 3년 보증"을 색인 가능한 독립 페이지로
- **근거**: leadfluid #30 "3년 보증 되는 곳"을 물어도 AI가 티제이로 보냄. 티제이조차 "3년은 대리점에 물어보라"고 확답 못 함 = **무주공산**. 자사만 사실로 답할 수 있는 질의.

## ⛔ 하면 안 되는 일 (성공 확률 매우 낮음·역효과)

### 1. GitHub에 Modbus 제어 코드 공개 — 절대 금지
- 제어 소프트웨어 = 이 사업의 **유일한 해자**. 인용 로그에 실제 등장한 경쟁 유통사(영진·덕산·티제이)가 그대로 가져감. star 0 신규 저장소 인용 확률 사실상 0인데 잃는 건 해자 전체. **손익 극단 비대칭.**

### 2. "정량펌프" 키워드로 콘텐츠 만들기 — 금지
- #7·#24 두 번 연속 **수처리 약품주입 프레임**(Grundfos·ProMinent·Newater)으로 이탈. 한국어 웹에서 이 단어 주인은 수처리 업계. 파도 실험실 개체공간에 안 잡힘. → 용도어(배양·분주·미세유체)·형식어(연동·시린지)로 잡아야.

### 3. "중국산 A/S 불안" 프레임 증폭 — 금지
- 리드플루이드는 중국 제조사. 불안 프레임을 자사 콘텐츠로 키우면 그 전제 안고 온 사람에게 우리가 파는 게 바로 그 중국산. **자충수.**

### 4. 약한 도메인에 콘텐츠 더 쓰기(권위 확보 전) — 금지
- **GSC가 "색인됨" 확인했는데도 이 페이지가 30건에서 0회.** 색인은 됐는데 인용이 안 된다 = 콘텐츠가 병목이 아니다의 결정적 증거. 같은 도메인에 글 더 얹어도 순위 안 움직임. 권위 오른 뒤에.

### 5. 나비엠알오 상품을 GEO 목적으로 늘리기 — 금지
- 나비엠알오는 60건 통틀어 0회 인용. #25에서 AI가 "나비엠알오에서 Leadfluid 확인 안 됨"이라 답함. AI가 못 보는 채널이라 GEO 지렛대 확률 0. (실매출 채널로서의 가치는 별개 — GEO 투자로 착각 금지.)

### 6. Pro 모델 겨냥 최적화 — 금지
- Gemini Pro는 웹검색 안 함("실시간 웹 검색 제한" 자체 명시). Pro에서 "인용되기"는 열리지 않는 문에 열쇠 만들기. Pro는 학습데이터 개체각인 싸움이지 인용 최적화 대상 아님.
