# AI 추천 탈환작전 3 — ChatGPT 실측 데이터 (2026-07-18)

실험실용 정량펌프(시린지/연동/기어/다이어프램/방폭) 구매 검토자가 실제로 검색할 만한 질문을 ChatGPT에 직접 입력하여, **어떤 브랜드가 추천되고 어떤 출처가 인용되는지**를 수집한 원본 기록.

## 2. 핵심 인사이트 (탈환 관점)

### 2-1. 웹검색 발동 여부가 곧 침투 가능성이다

25문항 중 **15문항에서 웹검색이 발동**했고, 10문항은 순수 모델 지식만으로 답했다.

- **웹검색 발동 문항(구매·가격·업체·브랜드평판·납기)** → 검색 인덱스에 콘텐츠를 올리면 **즉시 인용 가능**. 최우선 타겟.
- **웹검색 미발동 문항(원리·차이·트러블슈팅·구매가이드)** → 출처 인용이 0건이라 현재 아무도 점유하지 못한 상태. 콘텐츠를 쌓아 검색을 유발시키는 것이 2차 타겟.

### 2-3. 지도(로컬팩)가 답변 최상단에 뜬다 — 완전히 놓치고 있는 채널

질문 6("국내 정량펌프 제조사·대리점")에서 ChatGPT는 **본문보다 위에 구글 지도 로컬 결과를 통째로 붙였다.**

노출 업체: 이와끼코리아 / (주)제이이씨·Verder 한국지사(★5.0) / 코리아모터펌프(주) / 한일피엠(주)(★5.0) / (주)대영파워펌프(★5.0) / 동원펌프 — **별점·업종·영업시간까지 표시.**

→ 구글 비즈니스 프로필(업종을 "펌프 제조업체 / 산업용 장비 공급업체"로 등록 + 리뷰 확보)이 AI 답변 노출로 직결된다. 지금은 산업용 펌프 업체들이 실험실 펌프 질문까지 가져가고 있다.

### 2-5. LeadFluid 관련 실측 (가장 중요한 부분)

- **Q14 브랜드 평판**: "연구실에서 충분히 사용하는 브랜드". 장점으로 유량 재현성·모터 안정성·RS485/Modbus 제어를 꼽음. 단점으로 **"설명서 번역이 아쉽다", "국내 서비스망은 Longer보다 적은 경우가 있다"** 를 명시. 출처는 **leadfluid.com 본사 사이트뿐**, 한국 판매처 인용 0건.
- **Q16 한국 대리점**: "LeadFluid는 **한국 공식 총판을 홈페이지에 공개하지 않고 있다**. 국가별 대리점은 본사 문의 안내." → 본사 이메일(master@ / server1@leadfluid.com)을 안내. 국내 업체로는 **"정량펌프연구소"** 만 언급(출처: 실험셋업연구소), 그마저 "제조사 공식 독점 총판은 아니다"라고 단서를 붙임.
- **Q12 납기**: 국내 재고 보유 가능 브랜드 목록에서 **LeadFluid를 맨 앞에 배치** (LeadFluid > LongerPump > Watson-Marlow > KNF > Masterflex).
- **Q15 국내 가격**: LeadFluid BT50S 38~45만 / BT101S 50~60만 / BT100S 58~70만 / BT101L 87~100만 — **전부 캐시바이 인용**. "올포랩 = LeadFluid 공식 판매 제품 다수"로 기술.

→ **즉, ChatGPT는 LeadFluid를 좋게 평가하면서도 "한국 공식 대리점이 누군지 모른다". 이 공백을 메우는 것이 탈환의 1순위 과제.**

- **Q25 국산 브랜드**: ChatGPT가 **국산 정량펌프 업체를 단 한 곳도 이름을 대지 못했다.** ("국산 산업용 정량펌프 제조사", "주문형 OEM 제작 업체" 같은 무명 서술만 반복)

## 3. 실행 우선순위 제안

1. **"LeadFluid 한국 공식 대리점" 정체성을 웹에 명시** — 자사 사이트에 공식 대리점 선언 페이지(영문 병기), leadfluid.com 본사 대리점 페이지 등재 요청. 지금은 본사 이메일이 답변에 나가고 있다.
2. **가격 페이지를 모델별·숫자 노출로 공개** — 캐시바이/올포랩이 인용되는 이유는 단 하나, 가격 숫자가 크롤링되기 때문. BT50S/BT100S/BT101S/BT101L/WT600F 등 모델별 가격+재고 표기.
3. **구글 비즈니스 프로필 등록·리뷰 확보** — 업종을 펌프 제조/산업용 장비 공급업체로. Q6 유형 질문에서 지도가 본문보다 먼저 나온다.
4. **"국내 재고 즉시 출고" 를 전면에 배치** — 납기 질문에서 이미 LeadFluid가 1순위로 언급됨. 재고 리스트 페이지를 만들면 바로 인용 대상이 된다.
5. **"국내 A/S·부품 공급" 콘텐츠로 약점 반박** — ChatGPT가 반복 언급하는 단점이 정확히 "국내 서비스망 부족"과 "설명서 번역". 한글 매뉴얼 PDF + A/S 절차 페이지가 그대로 반박 근거가 된다.
6. **무주공산 기술 콘텐츠 선점** — 방폭등급/KCs, 유량 오차 원인·캘리브레이션(질량법), 튜브 재질별 수명표, 사양서 체크리스트, 맥동 저감. 출처 경쟁자가 0이라 진입 비용이 가장 낮다.
