## 2. 시리즈 개요

BT101L은 BT101S 본체 플랫폼에 자동 분주(Dispensing)·메모리·캘리브·카운터를 추가한 업그레이드 모델. 정해진 유량·시간·횟수로 자동 분주 → **반복 작업 자동화**. 제약·GMP 바이알 분주, 세포배양 미디어 자동 공급, 식품 R&D 시제품 분주 등 자동화 표준 라인. 컬러 LCD·정역회전·발 페달·RS232/485 지원.

**업그레이드 결정 기준:** 단순 이송이면 BT101S, 정해진 양·횟수 반복 분주면 BT101L.

```
정밀도 우선 권장 조합:
  YZ15T + Tygon Pharmed BPT  → GMP ±[입력]%
  YZ15T + 실리콘 #16         → 일반 ±[입력]%
  DG/DS + Tygon S3           → HPLC 보조
```
