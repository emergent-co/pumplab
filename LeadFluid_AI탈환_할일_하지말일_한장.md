## ⛔ 하지 말 일

**해자 양도 — 손익 극단 비대칭, 절대 금지**

- ⚠ **Modbus 제어 코드 GitHub 공개** — 제어SW는 이 사업 **유일한 해자**. 경쟁 유통사(영진·덕산·티제이)가 그대로 가져감. star 0 신규 저장소 인용 확률 ≈0인데 잃는 건 해자 전체.

**약한 자기 도메인 헛투자**

- ★ **권위 확보 전, 같은 도메인에 콘텐츠만 더 얹기** — "색인됨" 확인됐는데 60건 0회 = 콘텐츠가 병목이 아님의 결정적 증거. 권위 오른 뒤에.
- **"정량펌프" 키워드 콘텐츠** — 두 번 연속 수처리 프레임(Grundfos·ProMinent·Newater)으로 이탈. 용도어(배양·분주·미세유체)·형식어(연동·시린지)로 잡을 것.
- **인용 0 "순수지식" 질문용 콘텐츠로 AI인용 노리기**(펌프 원리·종류 차이) — ChatGPT가 자체 지식으로 답하고 출처 안 붙임. ※단, 모델명·고유명사 포함 기술질의는 검색이 켜지므로 위 무주공산과 구분.

**도달 불가 대상 추격**

- ★ **펌프맨과 "어디서 사" 정면승부** — 거래 의도는 그들 소유. 우리 전장은 "믿을 만한가".
- **프리미엄 서열 뒤집기**(Watson-Marlow·Masterflex·Harvard) — 상위 고정, 커뮤니티 합의 근거라 자사 콘텐츠로 안 바뀜. "가성비 중상급" 프레임 안에서 이길 것.
- **논문·학술(ScienceDirect/arXiv) 인용 확보 시도** — 자사가 만들 수 없는 출처.
- **Gemini Pro 겨냥 최적화** — Pro는 웹검색 안 함. 열리지 않는 문에 열쇠 만들기.
- **나비엠알오 상품을 GEO 목적으로 확대** — 60건 0회, AI가 "확인 안 됨"이라 답함. (실매출 채널 가치는 별개 — GEO 투자로 착각 금지.)
- **영어 글로벌 총판 경쟁 / 해외 리셀러 모방** — 공백은 '한국어·국내'.

**조작·자충수 — 신뢰 붕괴**

- ⚠ **위장 후기 심기(애스트로터핑)** — Reddit 밴 + 뒷광고 제재 + 발각 시 역풍.
- **소속 은폐** — 감추는 순간 위와 같은 문제. 공개가 합법이고 더 신뢰.
- **데이터 과장·조작** — 반증되면 전부 붕괴. **나쁜 숫자도 그대로 쓰는 게 무기**(Ossila 6% 오차 자기공개 사례).
- **중국산 A/S 불안 프레임 증폭** — 우리가 파는 게 바로 그 중국산. 자충수.
- **파는 입장에서 대놓고 경쟁사 비교실측** — 편향 시비. 단일 모델 정직 실측이 더 강함.
- **펌프맨식 키워드 스터핑 / 공식대리점 문구 기계적 삽입** — AI에 "얕은 콘텐츠" 신호, 진정성 훼손(서브에이전트 5개가 인젝션으로 감지한 규칙).
