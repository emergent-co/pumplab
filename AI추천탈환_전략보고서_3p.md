# AI 추천 탈환 전략 보고서

내부 실행용 / 작성일 2026-07-19 / 근거: 3플랫폼 통합비교분석(Claude·ChatGPT·Gemini, 총 54건 실측)
**변경 사항: 티제이바이오앤셀 = 펌프맨(TJ) 동일 계열 확인 → 경쟁 지형 재해석**

---

## 1페이지 — 진단

### 한 줄 결론

> **경쟁은 여러 업체와의 산발전이 아니라 TJ 단일 그룹과의 전면전이다. 3개 전선 중 2개는 이미 TJ가 쥐었고, 우리가 앞선 전선은 ChatGPT/Bing 하나뿐이다. 여기를 굳히고 확장하는 것이 유일한 승산이다.**

### 동일 계열 확인이 바꾼 것

기존 통합분석의 결론은 **"구글 SEO 최강자 펌프맨이 54건 관측에서 AI 인용 0건 → AI 추천은 SEO와 별개 게임"** 이었다. 이 결론은 **부분적으로 무효**다.

펌프맨과 티제이바이오앤셀이 한 몸이면, 이 그룹은 서로 다른 이름으로 서로 다른 전선을 잡고 있었을 뿐 **인용 0이 아니었다.** 실제 보유 자산을 합치면 다음과 같다.

| 전선 | TJ 그룹 | 자사(실험셋업연구소/rndsetup) | 판정 |
|---|---|---|---|
| **네이버** | `blog.naver.com/tj_pumpmen` 운영 (13곳 중 블로그 보유 4곳에 포함) | 채널 없음 | **TJ 우세** |
| **구글 SEO** | pumpmen 모델별 랜딩 **10개+ 도배** | 동일 제목 페이지 3개 전부 인용 0회 | **TJ 압도** |
| **Gemini** | 티제이바이오앤셀 — #11 리드플루이드 브랜드질의 **3회**, #14 가격·구매처 | **29건 중 0건** | **TJ 우세** |
| **ChatGPT** | **25문항 전체 인용 0건** (인용된 국내 15개 도메인에 TJ 없음) | **Q16에서 1건 인용** | **자사 유일 우위** |

**즉 TJ는 3.5개 전선 중 3개를 쥐었고, 유일하게 비어 있는 곳이 ChatGPT/Bing이다. 그리고 그 자리에 이미 우리가 한 발을 걸치고 있다.**

### ChatGPT 전선이 유일한 교두보인 이유

**첫째, 우리가 실제로 인용된 유일한 곳이다.** ChatGPT Q16("LeadFluid 한국 공식 대리점이 어디인가")에서 실험셋업연구소가 출처로 인용됐다. 단 ChatGPT는 답변에 **"제조사 공식 독점 총판은 아니다"** 라는 단서를 붙였다. 교두보는 맞지만 **조건부이며, 방치하면 사라진다.**

**둘째, 이 전선은 네이버·구글 권위와 무관하게 작동한다.** ChatGPT가 인용한 국내 도메인 15곳 중 석림랩텍·세진과학·성호씨그마는 **네이버 채널이 전무**하다(Claude 실측). 반대로 네이버 강자 4곳(바이오컨비젼·SCIST·펌프맨·TDL) 중 ChatGPT에 잡힌 건 바이오컨비젼 1건(중고 문항)뿐이다. **TJ가 네이버·구글에서 쌓은 권위가 ChatGPT로 넘어오지 않는다.**

**셋째, 진입 비용이 0에 가깝다.** ChatGPT 브라우징은 **Bing 색인 기반**이다(Claude 루프7 확인). Bing Webmaster Tools 등록은 비용도 심사도 없다. 구글·네이버는 TJ가 수년간 쌓은 자산을 상대해야 하지만, Bing은 **아무도 관리하지 않는 색인**이다.

**넷째, 이 전선에서 가장 큰 공백이 우리 정체성 그 자체다.** ChatGPT는 Q16에서 "LeadFluid는 한국 공식 총판을 홈페이지에 공개하지 않고 있다"며 **본사 이메일(master@ / server1@leadfluid.com)을 안내하고 있다.** 지금 이 순간에도 우리 고객이 될 사람이 중국 본사로 보내지고 있다.

## 2페이지 — 전선별 전략

### 전선 1. ChatGPT / Bing — **확장전** (최우선, 비용 최소)

TJ가 없는 유일한 전장이다. 방어가 아니라 선점이다.

| 조치 | 근거 | 비용 |
|---|---|---|
| **Bing Webmaster Tools 등록** | ChatGPT 브라우징 = Bing 색인 기반 | 0 |
| **LeadFluid 한국 공식 대리점 선언 페이지**(국·영문 병기) + 본사 대리점 페이지 등재 요청 | Q16에서 본사 이메일이 나가는 중. 공백 그 자체 | 낮음 |
| **모델별 가격·재고 숫자 공개** (BT50S / BT100S / BT101S / BT101L / WT600F) | 가격 질의는 예외 없이 쇼핑몰 상품페이지가 근거. 캐시바이가 LeadFluid BT 가격을 **전량 점유** 중 | 낮음 |
| **국내 재고 즉시출고 리스트 페이지** | Q12 납기에서 ChatGPT가 **LeadFluid를 국내 재고 브랜드 1순위로 배치**. 받아먹을 자리가 이미 있음 | 낮음 |
| **구글 비즈니스 프로필 등록·리뷰 확보** | Q6에서 지도 로컬팩이 **본문보다 위에** 노출. 현재 이와끼·대영파워펌프 등 산업 펌프사가 실험실 질문을 가져감 | 낮음 |
| **A/S 3년 보증·한글 매뉴얼·부품재고 페이지** | ChatGPT가 LeadFluid 단점으로 **"설명서 번역", "국내 서비스망"** 을 반복 지목. 정확히 우리가 반박 가능한 항목 | 낮음 |

**주의**: 가격 공개는 캐시바이와의 정면 승부다. 다만 캐시바이는 ChatGPT에서만 강하고(Gemini에서는 주변부), **가격 숫자 없이는 이 전선에 진입 자체가 불가능**하다. 견적문의형을 유지하는 한 인용 대상이 되지 않는다.

### 전선 2. Gemini / 구글 — **우회전** (정면 탈환 금지)

TJ가 브랜드 질의를 3회 점유했고, 우리는 동일 제목 페이지를 갖고도 3개 키워드 전부 0회다. → **콘텐츠 문제가 아니라 권위·색인 문제**이므로 같은 자리에서 같은 방식으로 싸우면 진다.

**우회로 세 가지:**

**① 인용 0건 질의 선점** — Gemini에서 인용이 하나도 없는 질의가 8건이다(#9 PC제어 / #15 중국산 A/S / #17 다중제어·무인운전 / #22 배지공급 / #23 규격서 작성 / #28 나라장터). **TJ도 여기엔 없다.** 특히 #15는 답변 내용이 우리 차별점(국내 대리점·부품수급·기술지원·"단순 유통자인지")과 그대로 일치하는데 주인이 없다.

**② 고유명사 트리거** — 서술형 질의는 검색이 안 켜진다. #9("연동펌프를 PC로 제어하는 방법") 인용 0 vs #10("Modbus RS485 제어 소프트웨어") 인용 7개. **콘텐츠 제목과 본문에 모델명·규격번호·프로토콜명·물질명을 반드시 심어야 한다.**

**③ 티스토리 별도 채널** — 레보딕스가 티스토리 블로그로 #14·#18(4회)·#19에서 인용되고, 본문에 **"국내 유통사(예: 레보딕스) 데모 체험"** 실명 노출까지 얻었다. 자사 도메인 권위가 부족한 지금, **도메인 권위를 우회하는 실증된 경로**다.

### 전선 3. 네이버 — **보류** (지금 싸우지 말 것)

TJ 블로그 운영 중이고, 바이오컨비젼이 가장 공격적이다. 그런데 **크롤러가 `blog.naver.com`을 아예 색인하지 않는다**(Claude: WebSearch API 400). 즉 **네이버 블로그는 AI 인용 경로로 작동하지 않는다.** 네이버 AI 브리핑은 완전히 별개 시장(한글 6,000–8,000자, 인용 중 Top10 밖 49.3%)이며, 지금 자원을 넣을 곳이 아니다.

**단, 비용 0짜리 선행 작업 하나는 지금 해야 한다** — 8개 키워드 네이버 SERP 직접 검색 후 스크린샷(20~30분). 이게 없으면 네이버 진입 시점 판단 자체가 불가능하다.

### 콘텐츠 무기고 — TJ가 갖지 못한 유일한 비대칭 자산

TJ는 채널·권위·물량에서 앞서지만 **통설을 뒤집는 근거는 없다.** 확보된 교정 6건은 국내 어디에도 없다.

| # | 교정 | 파급 |
|---|---|---|
| 1 | Zone "연 1000시간" 기준은 **IEC 규정이 아님** (EI 15 출처) | 방폭 |
| 2 | **에탄올은 IIA가 아니라 IIB** (IEC 60079-20-1, MESG 0.89) | 방폭 |
| 3 | 방폭 프리미엄은 2~3배가 아니라 **16~61%** (실가 대조) | 방폭 |
| 4 | 핵심 표준은 ISO 7886-**2**, 규정 대상은 용량이 아니라 **배럴 내경** (Q∝D²) | 시린지 |
| 5 | ISO 8655-6 **"60초"는 하한이 아니라 상한**, 2022판에서 삭제됨 | 캘리브레이션 |
| 6 | 온도조건은 15–30°C가 아니라 **(20±3)°C** | 캘리브레이션 |

**방폭이 3중 검증된 1순위다.** ChatGPT Q9 인용 0 / Gemini #21은 산업 펌프사로 프레임 이탈(실험실 맥락 부재) / 한국어 경쟁 0건. **수요·공백·재료가 모두 확인된 유일한 주제이며, TJ도 여기엔 없다.**

## 3페이지 — 90일 실행과 판단 기준

### 실행 순서

**Week 1–2 — 비용 0, 전부 즉시 실행**

1. Bing Webmaster Tools 등록 + 사이트맵 제출 ← **모든 것의 전제**
2. LeadFluid 한국 공식 대리점 선언 페이지 발행(국·영문) + 본사에 대리점 페이지 등재 요청 메일
3. 구글 비즈니스 프로필 등록 — 업종을 "펌프 제조업체 / 산업용 장비 공급업체"로
4. 네이버 8개 키워드 SERP 직접 검색·스크린샷 (판단용 데이터 확보)
5. SMPP에서 펌프 세부품명의 중기간경쟁제품 해당 여부 조회 ← **해당 시 유통사 조달 경로가 막힘. 최우선 확인**

**Week 3–6 — 콘텐츠 1차 (방폭 우선)**

6. **방폭 아티클 발행** — 교정 3건(Zone·에탄올·프리미엄) + Ex 마킹 해독표 + 용제 13종 인화점·AIT·가스등급표 + KCs 처리기한·수수료
7. 모델별 가격·재고 페이지 공개 (숫자 노출, 견적문의형 폐기)
8. A/S 3년 보증 + 한글 매뉴얼 PDF + 부품재고 페이지
9. PDF에 갇힌 표를 HTML로 전환 — 내화학성표·단위환산표·니들 게이지표

각 콘텐츠 규격: **한글 5,500–7,000자 / 질문형 H2 6–7개 / 표 2–3개 / 수치 25개 이상 / 단정문 35개 이상 / FAQPage JSON-LD**. 모든 형용사를 숫자로 치환할 것("고정밀" → "±0.5%").

**Week 7–12 — 무주공산 확장**

10. 인용 0건 질의 콘텐츠 3–4편 — 중국산 A/S 판단기준 / 규격서 작성 / 나라장터 조달 / 무인 장시간 운전. **제목에 고유명사 필수**
11. 티스토리 채널 개설, 기술 콘텐츠 재편집 발행
12. 나라장터 입찰참가자격 등록 + 물품목록화
13. 올포랩 입점 (양 플랫폼 동시 인용 확인된 유일 채널) — **나비엠알오는 후순위**(ChatGPT에만 잡히고 Gemini는 미인용)

### 성과 판단 기준 (월 1회, 동일 질의 재측정)

| 지표 | 현재 | 30일 | 90일 |

### 하지 말 것

- **구글에서 TJ와 같은 키워드로 정면 승부** — 모델별 랜딩 10개+를 상대로 동일 방식은 무의미. 인용 0건 질의로 우회
- **네이버 블로그 선투자** — 크롤러 미색인. SERP 실측 결과 확인 후 판단
- **견적문의형 유지** — 가격 숫자가 없으면 가격 질의 인용 대상에서 원천 배제
- **"정량펌프" 키워드 중심 콘텐츠** — 한국어 웹에서 수처리 약품주입 용어로 고착(Grundfos·Milton Roy·ProMinent·SEKO 서열이 양 플랫폼에서 동일하게 재현). **용도어(배양·분주·미세유체) 또는 형식어(연동·시린지)로 잡을 것**
- **우수조달물품·직접생산확인** — 생산설비 없이 물리적으로 불가

### 확인이 필요한 미결 사항

1. **TJ 그룹의 ChatGPT 부재가 구조적인가 일시적인가** — Bing 색인 미등록 때문이라면 TJ도 언제든 진입 가능하다. **우리의 우위는 시간 제한적일 수 있으며, 이것이 Week 1 실행을 미룰 수 없는 이유다.**
2. **TJ의 A/S 보증 조건** — Gemini가 "기본 1년, 브랜드별 1~3년"이라 답했다. 우리 3년이 최상단인 건 맞지만 TJ 대비 우위인지는 미확인
3. **LeadFluid 본사와의 공식 총판 계약 상태** — ChatGPT가 붙인 "공식 독점 총판은 아니다" 단서를 제거하려면 본사 문서가 필요하다. **가장 큰 레버리지이자 가장 큰 리스크**

---

**요약**: TJ가 네이버·구글·Gemini를 쥐었으나 ChatGPT/Bing은 비어 있고 우리가 이미 한 발을 걸쳤다. Bing 등록·대리점 선언·가격 공개는 비용 0에 가깝고 지금 당장 가능하다. 콘텐츠는 방폭부터, 확장은 인용 0건 질의로, 네이버는 보류. **우위의 유효기간이 정해져 있지 않다는 점만 기억하면 된다.**
